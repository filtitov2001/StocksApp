## Run locally

1. Install project from GitHub

```bash
  git clone https://github.com/filtitov2001/ToDoFirebaseApp/
```
2. Go to app folder
```bash
  cd StocksApp
```
3. Install dependency 
```bash
  pod install
```
4. Open project
```bash
  open StockApp.xcworkspace
```
